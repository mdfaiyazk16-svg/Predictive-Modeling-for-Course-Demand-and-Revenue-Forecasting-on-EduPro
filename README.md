# EduPro Predictive Modeling for Course Demand and Revenue Forecasting

## Project Overview

EduPro is an online learning platform that offers courses across multiple categories such as Programming, Machine Learning, Cybersecurity, Marketing, Finance, Design, and Project Management.

This project applies Machine Learning techniques to predict:

* Future Course Enrollment Demand
* Course Revenue
* Category-Level Revenue Trends

The objective is to transform historical course and transaction data into actionable business intelligence that helps EduPro make data-driven decisions regarding course launches, instructor onboarding, pricing strategies, and resource allocation.

---

# Problem Statement

EduPro currently lacks:

* Predictive models for course enrollment demand
* Revenue forecasting capabilities
* Data-driven support for pricing decisions
* Forecasting mechanisms for future course success

As a result, business decisions rely primarily on historical observations rather than predictive intelligence.

This project addresses these challenges using Machine Learning-based forecasting models.

---

# Project Objectives

The primary objectives of this project are:

1. Predict future enrollment demand for courses.
2. Forecast course-level revenue.
3. Analyze category-level revenue trends.
4. Identify key factors influencing revenue generation.
5. Build an interactive Streamlit dashboard for business stakeholders.

# Key Findings

* Course Price is the strongest revenue predictor.
* Instructor Expertise significantly influences revenue generation.
* Cybersecurity, Machine Learning, and Project Management courses generate higher revenue.
* Teacher experience contributes positively to course performance.
* Revenue forecasting achieved strong predictive performance compared to enrollment forecasting.
